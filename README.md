# Sparkfolio

The Sparkfolio Investment Framework is a comprehensive guide to strategic asset allocation for modern wealth resilience. It serves as an institutional-grade, yet beginner-friendly, playbook for structuring, managing, and optimizing your investment portfolio.

## Live Website
[https://sparkience-ai.github.io/sparkfolio/](https://sparkience-ai.github.io/sparkfolio/)

## Features
- **Core & Satellite Strategies:** Learn how to structure your portfolio with a strong foundation and high-alpha satellites.
- **Goal-Based Asset Allocation:** Clear guidelines mapping time horizons to asset classes.
- **The Retirement Playbook:** Deep dive into India's 3% Safe Withdrawal Rate, the Bucket Strategy, and Sequence of Returns Risk mitigation.
- **Behavioral Guardrails:** Science-backed strategies to prevent panic selling and emotional investing.
- **Tax-Aware Investing:** Explore strategies like Tax-Gain Harvesting, Asset Location (Tax Alpha), and NPS Tax Efficiency.
- **SEO & AI-Optimized:** Built with comprehensive JSON-LD structured data so AI assistants and search engines can expose high-value financial Q&As.

## Development
This project is built using HTML, CSS (Vanilla), and JavaScript, and bundled with Vite.

### Setup
```bash
npm install
npm run dev
```

### Build
```bash
npm run build
```
The site is built and pushed to GitHub Pages.
